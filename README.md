# biosquad
repository for UN5550 semester project, "bio squad" group

data has been acquired from the Multi-Resolution Land Characteristics (MRLC) Consortium and NOAA at (https://gml.noaa.gov/ccgg/trends/) for use in our data science project.


data is stored in the 'data' folder: biosquad/tree/main/data.

the subfolders 'gpp' and 'lcc' contain the data acquired from the MRLC and NOAA, respectively.

some figures are stored in the 'figures' folder: biosquad/tree/main/figures


our project notebook is saved in the main folder as a .ipynb file called *biosquad.ipynb*
